Platform Preparation
--------------------
Run the following commands:
  sudo dnf update -y
  sudo dnf install httpd mariadb-server php php-cli php-fpm php-mysqlnd php-gd php-mbstring php-xml php-opcache php-curl php-zip php-intl unzip git -y

Installation/Test Notes:
First and foremost installing this way does not include mod_ssl for apache that will need to be added:
  sudo dnf install mod_ssl -y

You will be missing wget, tar and nano
  sudo dnf install wget tar nano -y

These utilities are not required by GuildCMS itself.  They are recommended for server administration and installation convenience.

Rocky installs PHP 8.0.30 by default.  GuildCMS recommends PHP 8.3
Upgrading via DNF modules
  sudo dnf module list php
  sudo dnf module enable php:8.3 (might as well jump to the highest offered)
  sudo dnf install php-cli php-fpm
  sudo php -v
  sudo php-fpm -v
Upgrade was successful

Enable and start Apache, PHP-FPM and MariaDB
  sudo systemctl enable httpd php-fpm mariadb
  sudo systemctl start httpd php-fpm mariadb

Now you might have an issue with your ssl cert as its self signed
  https://yoursite/

Get the files
go into your domain folder (for a new server its /var/www/html/)
  cd /var/www/html
  sudo wget https://guildcms.theregs.org/downloads/guildcms.tar.gz
